# Logo ARES

## Conversão de SVG para PNG

Para converter o arquivo `ares.svg` para `ares.png`, você pode usar:

### Opção 1: Usando ImageMagick
```bash
convert ares.svg ares.png
```

### Opção 2: Usando Inkscape
```bash
inkscape ares.svg --export-filename=ares.png
```

### Opção 3: Online
- Abra o arquivo `ares.svg` em um navegador
- Tire um screenshot ou use ferramentas online de conversão

### Opção 4: Usando GIMP
- Abra o arquivo SVG no GIMP
- Exporte como PNG

## Estrutura do Logo
O logo ARES consiste em:
- 5 círculos interconectados em formato de estrela
- Cor: #20B2AA (azul-teal)
- Dimensões: 60x60 pixels
- Linhas de conexão entre os círculos

Após a conversão, o arquivo `ares.png` será usado automaticamente pelo relatório HTML.
