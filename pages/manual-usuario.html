1. Introdução ao LexiDecis

O que é o LexiDecis?

O LexiDecis é uma plataforma de inteligência artificial corporativa que oferece múltiplos modelos de IA para auxiliar em diversas tarefas profissionais. A plataforma integra diferentes provedores de IA como OpenAI, Gemini, Anthropic, DeepSeek, Groq e um motor de IA personalizado.

Principais Funcionalidades

  • Chat Inteligente: Conversas com múltiplos modelos de IA
  • Gerenciamento de GPTs: Seleção e configuração de diferentes modelos
  • Histórico de Conversas: Acesso a todas as conversas anteriores
  • Interface Responsiva: Funciona em desktop e dispositivos móveis
  • Sistema de Autenticação: Login seguro com Firebase
  • Painel Administrativo: Controle total para administradores

2. Primeiros Passos

Como Acessar o Sistema

  1. Acesse a URL do LexiDecis no seu navegador
  2. Na tela de login, insira seu email e senha
  3. Clique em "Entrar" para acessar a plataforma
  4. Aguarde o carregamento inicial do sistema
Dica: Na primeira vez que acessar, o sistema verificará o status de todos os serviços de IA. Aguarde a verificação ser concluída antes de começar a usar.

Configuração Inicial

Após o primeiro login, o sistema irá:

  • Carregar a lista de GPTs disponíveis
  • Selecionar automaticamente um GPT padrão
  • Inicializar o chatbot
  • Carregar seu histórico de conversas

3. Interface do Usuário

Layout Principal

Sidebar (Barra Lateral)

A sidebar contém todas as funcionalidades principais do sistema:

  • Botão de Seleção de GPT: Para escolher diferentes modelos de IA
  • Botão Nova Conversa: Para iniciar uma nova conversa
  • Campo de Pesquisa: Para buscar conversas antigas
  • Lista de Chats: Histórico de todas as suas conversas
  • Menu do Usuário: Configurações, perfil e logout

Área Principal

Área de Chat

A área principal é onde acontecem as conversas com a IA:

  • Mensagens: Exibição das mensagens trocadas
  • Campo de Entrada: Para digitar suas perguntas
  • Botões de Ação: Enviar, anexar arquivos, gravar áudio

Responsividade

Dispositivos Móveis: Em telas menores, a sidebar pode ser recolhida usando o botão no canto superior direito. Toque no botão para expandir/recolher a sidebar.

4. Sistema de Chat

Como Iniciar uma Conversa

  1. Clique no botão "Nova Conversa" na sidebar
  2. Digite sua pergunta no campo de entrada na parte inferior
  3. Pressione Enter ou clique no botão de enviar
  4. Aguarde a resposta da IA

Funcionalidades do Chat

Anexar Arquivos

Você pode anexar arquivos para que a IA analise:

  • Clique no ícone de anexo
  • Selecione o arquivo desejado
  • O arquivo será processado e enviado junto com sua mensagem

Gravação de Áudio

Para dispositivos com microfone:

  • Clique no ícone de microfone
  • Fale sua pergunta
  • Clique novamente para parar a gravação
  • O áudio será convertido em texto automaticamente
Dica: Use Shift + Enter para quebrar linha no campo de entrada. Use apenas Enter para enviar a mensagem.

5. Gerenciamento de GPTs

O que são GPTs?

GPTs (Generative Pre-trained Transformers) são diferentes modelos de inteligência artificial, cada um com características específicas e especializações diferentes.

Como Selecionar um GPT

  1. Clique no botão na sidebar
  2. Uma janela modal será aberta com todos os GPTs disponíveis
  3. Navegue pelas categorias ou use a busca para encontrar um GPT específico
  4. Clique no GPT desejado para selecioná-lo
  5. O GPT selecionado será aplicado à conversa atual

Tipos de GPTs Disponíveis

Categorias

  • OpenAI: GPT-4, GPT-3.5 e outros modelos da OpenAI
  • Google: Gemini Pro e outros modelos do Google
  • Anthropic: Claude e outros modelos da Anthropic
  • DeepSeek: Modelos especializados da DeepSeek
  • Groq: Modelos de alta velocidade da Groq
  • Motor de IA: Modelos personalizados e especializados
Importante: Cada GPT pode ter diferentes capacidades e limitações. Alguns são melhores para tarefas específicas como análise de código, redação criativa, ou análise de dados.

6. Gerenciamento de Chats

Lista de Chats

A sidebar mostra todas as suas conversas organizadas por data:

  • Hoje: Conversas do dia atual
  • Ontem: Conversas de ontem
  • Anteontem: Conversas de dois dias atrás
  • Datas anteriores: Conversas mais antigas

Como Gerenciar Chats

Buscar Chats

  1. Use o campo de pesquisa na sidebar
  2. Digite palavras-chave relacionadas ao chat
  3. Os resultados serão destacados automaticamente
  4. Clique no "X" para limpar a busca

Renomear Chats

  1. Clique no menu de três pontos (...) ao lado do chat
  2. Selecione "Renomear"
  3. Digite o novo nome
  4. Pressione Enter para confirmar

Excluir Chats

  1. Clique no menu de três pontos (...) ao lado do chat
  2. Selecione "Excluir"
  3. Confirme a exclusão na janela de confirmação
Atenção: A exclusão de chats é permanente e não pode ser desfeita. Certifique-se de que realmente deseja excluir a conversa antes de confirmar.

7. Gerenciamento de Perfil

Acessando o Perfil

  1. Clique no seu nome/avatar no final da sidebar
  2. No menu dropdown, clique em "Gerenciar Perfil"
  3. Uma janela modal será aberta com suas informações

Informações do Perfil

Dados Exibidos

  • Email: Seu endereço de email (somente leitura)
  • Nome de Exibição: Nome que aparece na interface
  • Senha: Opção para alterar sua senha

Alterando Informações

Como Editar

  1. Clique no campo que deseja editar
  2. Digite as novas informações
  3. Para alterar a senha, preencha os campos de senha atual e nova senha
  4. Clique em "Salvar Alterações"
  5. Aguarde a confirmação de sucesso
Dica: Mantenha seu nome de exibição atualizado para facilitar a identificação em ambientes corporativos.

8. Painel Administrativo

Acesso ao Painel Admin

Acesso Restrito: O painel administrativo está disponível apenas para usuários com privilégios de administrador.
  1. Clique no seu nome/avatar no final da sidebar
  2. No menu dropdown, clique em "Painel do Admin"
  3. Uma janela modal será aberta com o painel administrativo

Funcionalidades do Painel Admin

Gerenciamento de Usuários

  • Visualizar todos os usuários cadastrados
  • Gerenciar permissões e acessos
  • Monitorar atividade dos usuários

Gerenciamento de GPTs

  • Adicionar novos modelos de IA
  • Configurar parâmetros dos GPTs
  • Ativar/desativar modelos

Monitoramento do Sistema

  • Status dos serviços de IA
  • Métricas de uso
  • Logs do sistema

9. Dicas e Truques

Atalhos de Teclado

  • Enter - Enviar mensagem
  • Shift + Enter - Quebrar linha
  • Ctrl/Cmd + K - Focar no campo de busca
  • Escape - Fechar modais

Dicas para Melhores Resultados

Como Fazer Perguntas Efetivas

  • Seja específico: Quanto mais detalhada a pergunta, melhor a resposta
  • Use contexto: Mencione o contexto da sua dúvida
  • Faça uma pergunta por vez: Evite múltiplas perguntas em uma só mensagem
  • Especifique o formato: Se quiser uma resposta em formato específico, mencione isso

Trabalhando com Arquivos

  • Formatos suportados: PDF, DOC, DOCX, TXT, imagens
  • Tamanho máximo: Verifique os limites do sistema
  • Para análise de código: Use arquivos de texto simples
  • Para imagens: A IA pode descrever e analisar o conteúdo
Dica Pro: Use diferentes GPTs para diferentes tipos de tarefas. Por exemplo, use GPTs especializados em código para programação e GPTs gerais para redação criativa.

10. Suporte e Contato

Problemas Comuns

Solução de Problemas

Chat não carrega
  • Verifique sua conexão com a internet
  • Recarregue a página (F5)
  • Limpe o cache do navegador
  • Verifique se todos os serviços estão online
Resposta lenta da IA
  • Alguns modelos podem ser mais lentos que outros
  • Tente usar um GPT diferente
  • Verifique se há muitos usuários simultâneos
Erro de autenticação
  • Faça logout e login novamente
  • Verifique se suas credenciais estão corretas
  • Entre em contato com o administrador se o problema persistir

Como Obter Ajuda

Canais de Suporte

  • Documentação: Este manual e documentação técnica
  • Administrador: Entre em contato com o administrador do sistema
  • Logs: Verifique os logs do sistema para erros específicos
Informações Importantes:
  • O sistema monitora automaticamente o status de todos os serviços
  • Notificações aparecem quando há problemas de conectividade
  • Seus dados são armazenados de forma segura
  • O sistema possui backup automático das conversas
